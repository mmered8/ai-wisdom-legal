# Privacy Policy — AI Wisdom Stories

**Last updated:** March 29, 2026

## Overview

AI Wisdom Stories is a personal content creation tool operated by Mason Meredith. This policy describes how the application handles data.

## Data Collection

This application does not collect, store, or process any personal data from viewers or third parties. It is a content publishing tool, not a user-facing service.

## OAuth Tokens

The application stores OAuth tokens locally on the operator's machine in an encrypted configuration file. These tokens are used solely to authenticate with YouTube, TikTok, and X.com APIs for video publishing. Tokens are never shared or transmitted to third parties.

## AI-Generated Content

Videos are generated using AI services (xAI Grok). Prompts and generated content are processed through xAI's API subject to their privacy policy. No personal data is included in prompts.

## Third-Party Services

- **YouTube Data API v3** — for video uploads (subject to Google Privacy Policy)
- **TikTok Content Posting API** — for video uploads (subject to TikTok Privacy Policy)
- **X.com API v2** — for video posts (subject to X.com Privacy Policy)
- **xAI Grok API** — for content generation (subject to xAI Privacy Policy)
